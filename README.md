
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Database

I am using sqlite.

To get one locally, create the `.env` file and inside a variable with the name `DB_NAME`

## Cookie Key

In the `.env` file create a variable with the name `COOKIE_KEY`

## Example .env

```bash
DB_NAME=test.sqlite
COOKIE_KEY=cookieTestingTest
```
